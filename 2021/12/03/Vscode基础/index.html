Vscode基础

发表于 2021-12-03 更新于 2021-12-09 分类于 Vscode 阅读次数：
本文主要介绍vscode的有关快捷键，基础设置等，安装的话，官方下载包，默认向下一直next即可，在此不再赘述。

Vscode 快捷键

主命令（万能键）

  • F1或Ctrl+Shift+P:开启命令面板，可以输入任何命令。
  • Ctrl+P：根据名称查找文件，如下图。当前模式下输入 > 可以进入 Ctrl+Shift+P 模式
  • Ctrl+P模式下可以通过输入符号来进行快捷方式查找转换,其中输入”?”,列出当前可执行的操作

常用快捷键

  • Ctrl+N： 新建文件
  • Ctrl+Shift+N：打开一个新的VSCode编辑器
  • Ctrl+O：打开文件
  • Ctrl+Shift+O：打开文件夹
  • Ctrl+W：关闭当前文件
  • Ctrl+Shift+W 或 Alt+F4：关闭当前VSCode编辑器
  • Ctrl+\：新建窗口显示代码（相当于复制当前代码到一个新的窗口；同一引用，修改一个文件，其他相同文件会一起改变）
  • Ctrl+Alt+→：移动当前文件到右窗口，若没有右窗口，则创建一个新窗口
  • Ctrl+Alt+←：移动当前文件到左窗口
  • Ctrl+Tab：切换文件窗口
  • Ctrl+B：显示/隐藏侧边栏
  • Ctrl+`：显示/隐藏控制面板（Terminal）（反引号位置：英文输入法状态下，键盘ESC按键下面的按键）
  • Ctrl +/- ：放大/缩小编辑器窗口
  • F11：全屏显示
  • Ctrl+Shift+E：文件资源管理器（Explorer）
  • Ctrl+Shift+G：git管理窗口（Source Control）
  • Ctrl+Shift+X：扩展（插件）管理窗口（Extentions）
  • Ctrl+Alt+鼠标左键: 像vs一样选中列操作
  • 双击变量，右键‘更改所有匹配项’或Ctrl+F2：将自动高亮的变量、字符一次性替换。
  • Ctrl+Delete：删除当前行光标后的所有内容（字符串）
  • Shift+Delete:删除当前行
  • F12： 跳转到定义
  • Ctrl+Enter:光标跳转到新建行
  • Ctrl+/:注释
  • Shift+F12:列出所有定义

光标相关

  • 移动到行首：Home
  • 移动到行尾：End
  • 移动到文件结尾：Ctrl+End
  • 移动到文件开头：Ctrl+Home
  • 移动到后半个括号 Ctrl+Shift+]
  • 选中当前行Ctrl+i（双击）
  • 选择从光标到行尾Shift+End
  • 选择从行首到光标处Shift+Home
  • 删除光标右侧的所有字Ctrl+Delete
  • Shrink/expand selection： Shift+Alt+Left和Shift*+Alt+Right
  • Multi-Cursor：可以连续选择多处，然后一起修改，Alt+Click添加cursor或者Ctrl+Alt+Down 或 Ctrl+Alt+Up
  • Ctrl+D下一个匹配的也被选中
  • 回退上一个光标操作Ctrl+U
  • Ctrl+Up/Down 类似于滑轮的工作，文章上下移动。
    向上向下复制一行： Shift+Alt+Up或Shift+Alt+Down

配置Python环境

  1. 确保系统中已经安装Python，并设置了系统环境变量；
  2. 在VScode扩展中安装Python扩展；
  3. 为Python代码选择合适的解释器，打开命令面板,在命令行键入Python: Select Interpreter，选择合适的解释器

配置Python debug环境

在配置python 调试环境之前，先必须明确两个概念：

  1. 单次运行一个脚本视为一个task,相应的配置文件为tasks.json;

  2. 整个文件夹或者多个文件夹视为一个工作空间，配置文件为settings.json,而调试环境的配置文件叫launch.json

  3. debug前，修改launch.json,生成默认文件，然后根据所需要的参数配置即可，详情可参见https://www.xliam.com/2017/04/29/VSCode-Debug-Python/

vscode配置代码注释模板

选中首选项，选择用户片段，选择新建snippet或者编辑已有的。详细可参照https://blog.csdn.net/u011127019/article/details/73461831
